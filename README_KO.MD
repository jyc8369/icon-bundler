# Icon Bundler

`PNG` 또는 `JPG` 이미지를 하나의 `.ico` 파일로 변환하는 간단한 데스크톱 앱입니다.

생성되는 아이콘에는 다음 해상도가 포함됩니다.

- `16x16`
- `32x32`
- `48x48`
- `256x256`

원본 이미지는 비율을 유지한 채 들어가고, 남는 영역은 투명 배경으로 채워집니다.

## 다운로드해서 사용하는 방법

직접 빌드할 필요가 없습니다.

1. 이 저장소의 GitHub Releases 페이지를 엽니다.
2. 자신의 운영체제에 맞는 파일을 다운로드합니다.
   - Windows: `IconBundler-<버전>-windows.zip`
   - macOS arm64: `IconBundler-<버전>-macos-arm64.zip`
   - macOS x86_64: `IconBundler-<버전>-macos-x86_64.zip`
3. 다운로드한 압축 파일을 풉니다.
4. 압축을 푼 폴더 안에서 앱을 실행합니다.
5. `PNG`, `JPG`, `JPEG` 이미지를 선택합니다.
6. `ICO 변환` 버튼을 누릅니다.

변환된 `.ico` 파일은 원본 이미지와 같은 폴더에, 같은 이름으로 저장됩니다.

## 앱 안에서 할 수 있는 것

- 언어를 `en` / `ko` 중에서 선택할 수 있습니다
- 첫 실행 기본값은 `en` 입니다
- 파일 선택 버튼으로 원본 이미지를 고릅니다
- 출력 `.ico` 파일은 원본 이미지 옆에 생성됩니다

## 지원 파일 형식

- `.png`
- `.jpg`
- `.jpeg`

## 참고 사항

- 같은 이름의 `.ico` 파일이 있으면 덮어쓰기 전에 확인합니다.
- 너무 작거나 손상된 이미지는 변환에 실패할 수 있습니다.
- Windows용 파일은 Windows에서, macOS용 파일은 macOS에서 사용해야 합니다.
