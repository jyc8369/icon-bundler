# Icon Bundler

`PNG` 또는 `JPG` 이미지를 Windows용 `ICO` 파일 또는 macOS용 `ICNS` 파일로 변환하는 데스크톱 앱입니다.

앱에는 아래 두 개의 출력 버튼이 있습니다.

- `ICO 변환`
- `ICNS 변환`

원본 이미지는 비율을 유지한 채 들어가고, 남는 영역은 투명 배경으로 채워집니다.

## 다운로드해서 사용하는 방법

직접 빌드할 필요가 없습니다.

1. 이 저장소의 GitHub Releases 페이지를 엽니다.
2. 운영체제에 맞는 압축 파일을 다운로드합니다.
   - Windows: `IconBundler-<버전>-windows.zip`
   - macOS arm64: `IconBundler-<버전>-macos-arm64.zip`
   - macOS x86_64: `IconBundler-<버전>-macos-x86_64.zip`
3. 압축 파일을 풉니다.
4. 압축을 푼 폴더 안에서 앱을 실행합니다.
5. `PNG`, `JPG`, `JPEG` 이미지를 선택합니다.
6. `ICO 변환` 또는 `ICNS 변환` 버튼을 누릅니다.

변환된 파일은 원본 이미지와 같은 폴더에, 같은 이름과 확장자로 저장됩니다. 원본 이미지를 선택하면 ICO와 ICNS 출력 후보 경로가 함께 표시되어 변환 전에 저장 위치를 확인할 수 있습니다.

## 앱에서 동작하는 방식

- 파일 선택 버튼으로 원본 이미지를 고릅니다
- 언어는 `en` / `ko` 중에서 선택할 수 있습니다
- 첫 실행 기본값은 `en` 입니다
- 언어를 바꾸면 현재 창의 텍스트가 즉시 새 언어로 바뀝니다
- 패키징된 앱에는 아이콘이 포함됩니다

## 내보내기 해상도

ICO 내보내기는 다음 해상도를 사용합니다.

- `16x16`
- `32x32`
- `48x48`
- `256x256`

ICNS 내보내기는 원본보다 큰 크기가 나오기 전까지만 순서대로 저장합니다.

- `16x16`
- `32x32`
- `64x64`
- `128x128`
- `256x256`
- `512x512`
- `1024x1024`

## 지원 파일 형식

- `.png`
- `.jpg`
- `.jpeg`

## 참고 사항

- 같은 이름의 파일이 있으면 덮어쓰기 전에 확인합니다.
- 너무 작거나 손상된 이미지는 변환에 실패할 수 있습니다.
- Windows용 파일은 Windows에서, macOS용 파일은 macOS에서 사용해야 합니다.
- macOS 패키지는 zip 안에 `.app` 번들 형태로 들어 있습니다.
